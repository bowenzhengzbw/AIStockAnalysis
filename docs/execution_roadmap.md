## 依赖与风险提示
- **数据获取风险**：第三方接口策略变更 → 预留备用渠道（官方数据、网页备份）。
- **模型偏差风险**：建立人工复核机制，设定容错阈值，保留回滚策略。
- **安全合规风险**：所有付费数据按协议使用；日志与密钥管理遵循 Phase 0 安全基线。
- **交付节奏风险**：若出现重大需求变更，采用“预览→迭代”机制，确保每个模块可独立验收。
