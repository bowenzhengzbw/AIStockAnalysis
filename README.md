# 离线股票宏观分析仪表盘

该项目实现了一个在无外网环境也可运行的宏观-行业-公司分析仪表盘。所有用到的市场、板块与公司数据均预先缓存在 `data/offline_data.json` 中，避免了运行时访问互联网或安装第三方数据源的需求。

## 主要特性
- **离线宏观信号**：直接读取缓存的标普500、VIX 与 10 年期国债收益率指标，给出趋势、收益与波动判断。
- **板块景气评估**：基于缓存 ETF 表现数据衡量目标行业的相对强弱。
- **公司基本面速览**：展示估值、盈利、资本效率、杠杆、现金流与股东回报指标，并自动生成文字解读。
- **零外部依赖**：使用 Python 标准库构建简易 Web 服务，无需 pip 安装额外包。

## 快速开始
1. **启动服务**
   ```bash
   python app.py
   ```
   控制台将提示访问地址（默认 `http://0.0.0.0:5000`）。

2. **在浏览器访问**
   打开提示的地址，输入离线数据文件中已缓存的股票代码（默认示例包含 `AAPL`、`MSFT`、`TSLA`）。

3. **无需启动服务器的静态预览**
   如果只想快速感受页面布局，可直接在浏览器中打开仓库根目录下的 `preview.html`。该文件预先渲染了 AAPL 的示例分析，并复用了项目的样式文件以确保所见即所得。

4. **扩展离线数据**
   - 编辑 `data/offline_data.json`，在 `companies` 节点新增目标公司的基础信息；
   - 视需要补充 `macro` 与 `sectors` 的指标快照，以保持分析的一致性；
   - 保存后无需重启服务即可加载最新数据（请求时将重新读取文件）。

## 自定义与维护建议
- 若需要更新指标数值，可通过任意具备网络的机器收集数据后手动同步到离线文件。
- 可以为不同部门或场景准备多份数据文件，并在部署时通过环境变量 `PORT` 指定服务端口。
- 项目无第三方依赖，但仍建议在虚拟环境中运行，以便未来扩展功能时隔离环境。

欢迎根据自身投研框架继续完善数据与展现内容，使该离线工具更贴近实际需求。
